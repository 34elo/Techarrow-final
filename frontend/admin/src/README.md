# Каталог `src/`

| Слой | Папка | Назначение |
|------|-------|------------|
| App | `app` | Маршруты, провайдеры, стили |
| Widgets | `widgets` | Крупные UI-блоки страниц |
| Features | `features` | `auth`, `nav-tabs`, `quests`, `reports`, `reject` (диалог отклонения) |
| Entities | `entities` | Модели и типы домена |
| Shared | `shared` | API, конфиг, UI-kit, сторы, React Query, i18n |

Подробнее: [docs/architecture.md](../docs/architecture.md), [README в корне](../../README.md).
